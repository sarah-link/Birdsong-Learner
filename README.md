# Overview

Welcome to Birdsong Learner! This is a quick React webpage built to help you learn birdcalls by
giving you a call to listen to and letting you guess what bird it is. There are various
skill levels available.

## Installation

Birdsong Learner uses React

[Install Node.js and npm](https://nodejs.org/en/)

Install react-router-dom

`npm install react-router-dom`

Install react-bootstrap

`npm install react-bootstrap bootstrap`
